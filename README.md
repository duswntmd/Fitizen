## 정재호

### 인증 게시판
-  인증사진 업로드 정해진 날짜에 
-  인증사진 삭제
-  인증사진 인증해주기  이미했는가 
-  댓글 수정
### 마이페이지 
-  챌린지 달성도 표시 
### 채팅 
-  보낸 날짜 
-  전체 페이지에서 채팅
-  css 수정   
-  인증사진 올리기 정해진 날짜에
### 챌린지 수정페이지 
-  인증 날짜, 인증 방법만 수정 가능 
-  기간 전에만  기간중이면 불가능
-  삭제:기간 전에만 기간중이면 불가능 
### 인증사진 삭제
- 인증사진 몇개 ,업로드 요일에 따른 , 달성도 및 포인트 지급  채팅
- 인증사진 업로드 기간이 아닙니다
